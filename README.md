🏥 MediCure – Hospital-and-Patient Management System

MedicureSoftw is a Java-based Hospital Management System developed using Java Swing and Hibernate. The application streamlines hospital operations by providing modules for patient management, doctor management, billing, prescriptions, visitor passes, and administrative tasks.
The system is designed to simplify healthcare administration through an intuitive desktop interface and efficient database management.

****Features****
1.Secure Login System
2.Admin Dashboard
3.Patient Management
4.Doctor Management
5.Patient Admission
6.Bed Allocation
7.Prescription Management
8.Billing System
9.Patient Reports
10.Visitor Pass Management
11.Complaint Management
12.Database Integration using Hibernate
13.Java Swing GUI
